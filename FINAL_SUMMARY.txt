## How It Works

### For Users (Business)
```
1. Download QZ Tray (5 mins)
   └─ https://qz.io/download/

2. Configure Printer (5 mins)
   ├─ Open KIFSHOP Treasury
   ├─ Click "Imprimante"
   ├─ Select printer
   └─ Click "Test impression"

3. Accept Payment (0 clicks)
   ├─ Add items
   ├─ Process payment
   ├─ Receipt prints AUTOMATICALLY
   └─ Drawer opens AUTOMATICALLY

Total: 10 minutes setup, then fully automatic! ✅
```

## System Architecture

```
┌─────────────────────────────────────┐
│   KIFSHOP Browser App               │
│   (Next.js 16 + React 19)           │
└────────────┬────────────────────────┘
             │
             │ WebSocket
             │ localhost:8181
             ▼
┌─────────────────────────────────────┐
│   QZ Tray Desktop App               │
│   (Free application)                │
└────────────┬────────────────────────┘
             │
      ┌──────┴──────────┐
      │                 │
   USB │              Network
      │              TCP:9100
      ▼                 ▼
┌──────────────┐  ┌──────────────┐
│ POS80 Printer│  │ POS80 Printer│
│   (USB)      │  │  (Network)   │
└──────────────┘  └──────────────┘
      │                 │
      └────────┬────────┘
               │
        ┌──────┴──────┐
        ▼             ▼
    Receipt       Cash Drawer
    Output         Opens!
```
