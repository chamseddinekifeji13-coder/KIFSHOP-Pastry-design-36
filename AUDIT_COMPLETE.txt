## 🔴 5 ANOMALIES CRITIQUES IDENTIFIÉES

### 1. **RLS Permissive sur Clients** (CRITIQUE)
- **Problème:** `USING (true)` = tout le monde peut voir
- **Conséquence:** Données multi-tenant exposées
- **Fix:** audit-002 → RLS tenant-safe

### 2. **Incompatibilité UUID/TEXT** (CRITIQUE)
- **Problème:** `clients.tenant_id` est UUID mais `tenants.id` est TEXT
- **Conséquence:** Clients ne peuvent pas être créés
- **Fix:** audit-002 → Conversion UUID→TEXT

### 3. **Tables Métier Manquantes** (CRITIQUE)
- **Tables:** suppliers, raw_materials, recipes, orders, stock_movements, etc.
- **Conséquence:** Système non-fonctionnel
- **Fix:** audit-003 → Création complète

### 4. **RLS Permissive Best Delivery** (ÉLEVÉ)
- **Problème:** Données exposées entre tenants
- **Conséquence:** Fuite support_tickets, sales_channels
- **Fix:** audit-004 → RLS correcte

### 5. **Indexes Manquants** (ÉLEVÉ)
- **Problème:** Performance dégradée
- **Conséquence:** Requêtes lentes
- **Fix:** Tous les scripts → Indexes optimisés
